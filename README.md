
# Advent of Code 2025

Solutions in C++ for the daily challenges of [Advent of Code 2025](https://adventofcode.com/).

This repository contains one folder per event day, each with its own source code, input, and output files. The Makefile allows you to easily compile and run any day by setting the `DAY` variable.

- Language: C++
- Easy compilation and execution with Makefile
- Organized by day
